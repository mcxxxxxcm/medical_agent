## 外伤急救与处理指南

### 01 软组织损伤

软组织损伤是指皮肤、肌肉、肌腱、韧带等组织的损伤，是最常见的外伤类型。

**分类：**
- 挫伤：钝性外力导致的皮下组织损伤，表现为局部肿胀、疼痛、皮下淤血
- 擦伤：皮肤表层因摩擦导致的损伤，表现为表皮破损、少量出血
- 撕裂伤：锐器或钝力导致的皮肤及皮下组织撕裂，伤口边缘不整齐
- 切割伤：锐器导致的整齐切割伤口，出血较多但组织损伤相对较轻

**急救处理（RICE原则）：**
1. Rest（休息）：立即停止活动，避免加重损伤
2. Ice（冰敷）：伤后24-48小时内，每次冰敷15-20分钟，间隔1-2小时，有助于减轻肿胀和疼痛
3. Compression（加压包扎）：使用弹性绷带适度加压包扎，注意不要过紧影响血液循环
4. Elevation（抬高）：将受伤部位抬高至心脏水平以上，促进静脉回流，减轻肿胀

**注意事项：**
- 伤后48小时内禁止热敷，热敷会加重出血和肿胀
- 禁止在伤处涂抹红花油等活血药物揉搓，会加重出血
- 如出现明显畸形、骨擦音、异常活动，可能合并骨折，应立即就医
- 开放性伤口需在6-8小时内清创缝合

### 02 关节扭伤

关节扭伤是指关节周围韧带因过度牵拉导致的损伤，常见于踝关节、膝关节、腕关节和手指关节。

**分级：**
- I度（轻度）：韧带少量纤维撕裂，关节稳定，轻度肿胀和压痛
- II度（中度）：韧带部分撕裂，关节轻度不稳，中度肿胀和疼痛
- III度（重度）：韧带完全断裂，关节明显不稳，严重肿胀和功能障碍

**常见扭伤处理：**

**踝关节扭伤：**
- 立即停止行走，避免负重
- 冰敷15-20分钟，每2小时一次
- 用弹性绷带从足趾向小腿方向缠绕固定
- 抬高患肢，高于心脏水平
- 48小时后可开始温水浸泡和轻度活动
- 严重扭伤需X光检查排除骨折

**膝关节扭伤：**
- 立即停止活动，避免负重
- 冰敷并加压包扎
- 使用拐杖辅助行走
- 如出现关节内积血（关节迅速肿胀），可能为交叉韧带损伤，需急诊就医
- 膝关节扭伤后应避免过早负重和剧烈活动

**腕关节扭伤：**
- 停止使用患手
- 冰敷并使用腕关节护具固定
- 避免旋转和负重动作
- 如疼痛持续超过1周，需排除舟骨骨折

### 03 骨折

骨折是指骨的完整性或连续性遭到破坏，是较严重的外伤。

**分类：**
- 闭合性骨折：骨折处皮肤完整，骨折端不与外界相通
- 开放性骨折：骨折处皮肤破损，骨折端与外界相通，感染风险高

**常见骨折类型：**
- Colles骨折：桡骨远端骨折，多因手掌撑地摔倒所致，表现为"餐叉样"畸形
- 踝关节骨折：多因扭伤所致，表现为踝关节肿胀、畸形、无法负重
- 锁骨骨折：多因肩部着地摔倒所致，表现为肩部下垂、局部肿胀畸形
- 肋骨骨折：多因胸部撞击所致，表现为深呼吸和咳嗽时疼痛加重

**急救处理：**
1. 保持冷静，不要随意搬动伤者（特别是脊柱骨折）
2. 开放性骨折：用清洁敷料覆盖伤口，不要将骨折端推回伤口内
3. 固定：用夹板或就地取材（木板、硬纸板）固定骨折部位上下两个关节
4. 止血：如有出血，在伤口近心端加压包扎止血
5. 就医：所有骨折均需就医，进行X光检查和专业处理

**骨折警示征（需立即就医）：**
- 患肢明显畸形或缩短
- 开放性伤口伴骨外露
- 患肢远端感觉减退或脉搏消失（提示血管神经损伤）
- 胸部外伤后呼吸困难（提示肋骨骨折合并气胸）
- 颈背部外伤后四肢感觉异常（提示脊柱损伤）

### 04 烧烫伤

**分级：**
- I度烧伤：仅伤及表皮，皮肤发红、疼痛，无水疱，3-5天可愈合
- 浅II度烧伤：伤及真皮浅层，有水疱，基底潮红，剧痛，1-2周愈合
- 深II度烧伤：伤及真皮深层，水疱较小，基底苍白，痛觉迟钝，3-4周愈合
- III度烧伤：伤及全层皮肤，呈焦痂状，无痛觉，需手术植皮

**急救处理（冲脱泡盖送）：**
1. 冲：立即用流动冷水冲洗伤处15-20分钟，降低皮肤温度
2. 脱：在冷水中小心脱去伤处衣物，避免强行撕扯
3. 泡：将伤处浸泡在冷水中10-30分钟
4. 盖：用清洁的纱布或毛巾覆盖伤口
5. 送：严重烧烫伤立即送医

**注意事项：**
- 不要在伤口上涂抹牙膏、酱油、香油等，会增加感染风险
- 不要挑破水疱，水疱皮是天然的保护层
- 不要用冰块直接接触伤口，会造成冻伤
- 化学烧伤需大量清水冲洗至少20分钟
- 电击伤需检查心电图，即使表面伤不重也可能有心脏损伤

### 05 头部外伤

头部外伤是常见的外伤类型，需特别注意颅内损伤的可能。

**轻度头部外伤：**
- 表现：头皮血肿、头痛、头晕、恶心
- 处理：观察24-48小时，冰敷消肿，避免剧烈活动
- 可服用对乙酰氨基酚缓解头痛，避免使用阿司匹林（增加出血风险）

**需立即就医的警示征：**
- 意识丧失或意识模糊
- 持续加重的头痛
- 反复呕吐
- 鼻腔或耳道流出清亮液体（提示脑脊液漏）
- 瞳孔不等大
- 抽搐发作
- 肢体无力或麻木
- 行为异常或言语不清

**头皮血肿处理：**
- 伤后立即冰敷，每次15-20分钟
- 不要用力揉搓血肿处
- 大血肿需就医排除颅骨骨折
- 婴幼儿头皮血肿需特别注意，可能合并颅内出血

### 06 出血与止血

**出血分类：**
- 动脉出血：血液鲜红，呈喷射状，出血量大，最危险
- 静脉出血：血液暗红，呈涌出状，出血量中等
- 毛细血管出血：血液暗红，呈渗出状，可自行止血

**止血方法：**
1. 直接压迫止血：用清洁纱布直接压在伤口上，保持5-10分钟
2. 加压包扎止血：在伤口上放置敷料后用绷带加压包扎
3. 指压止血：在出血点近心端动脉搏动处用手指按压
4. 止血带止血：仅用于四肢大动脉出血且其他方法无效时，记录使用时间，每40分钟放松1-2分钟

**注意事项：**
- 不要在伤口内填塞异物
- 鼻出血时不要仰头，应低头用手指捏住鼻翼压迫止血
- 如伤口内有异物（如玻璃碎片），不要自行拔出，应就医处理
- 大量出血或止血困难时立即拨打急救电话

### 07 常见外伤用药

**外用消毒：**
- 碘伏：适用于皮肤和黏膜消毒，刺激性小，为首选
- 75%酒精：适用于完整皮肤消毒，有刺激性，不适用于伤口内
- 双氧水：适用于污染伤口清洗，有杀菌和去污作用
- 生理盐水：适用于伤口冲洗，无刺激性

**止痛药物：**
- 布洛芬：适用于外伤后疼痛和炎症，成人每次200-400mg，每日不超过1200mg
- 对乙酰氨基酚：适用于轻中度疼痛，成人每次300-600mg，每日不超过2000mg
- 注意：有胃溃疡病史者避免使用布洛芬，可选用对乙酰氨基酚

**外用药物：**
- 莫匹罗星软膏：适用于皮肤感染预防
- 红霉素软膏：适用于小伤口抗感染
- 云南白药：适用于闭合性损伤，开放性伤口慎用
- 活血化瘀类药物（如红花油）：仅适用于伤后48小时以上的闭合性损伤

### 08 何时需要就医

以下情况应立即就医：
1. 开放性骨折或关节脱位
2. 伤口深且出血不止
3. 头部外伤伴意识改变
4. 疑似内脏损伤（腹部外伤后持续疼痛）
5. 烧烫伤面积大于手掌或为深度烧伤
6. 动物咬伤（需注射狂犬疫苗）
7. 伤口内残留异物
8. 外伤后出现发热、红肿加重等感染征象
9. 扭伤后关节明显不稳或无法负重
10. 胸部外伤后呼吸困难

以下情况可先自行处理，观察后决定是否就医：
1. 小面积浅表擦伤
2. 轻度软组织挫伤（无骨折征象）
3. I度烧烫伤
4. 轻度关节扭伤（I度）
5. 小面积头皮血肿（无意识障碍）
